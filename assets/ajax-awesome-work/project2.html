pAwlert
by Kaely Mills
For our final project in UCF’s Coding Bootcamp, my classmate and I built a web application to help pet owners find their lost pets. The name pAwlert is our made-up term that represents the missing paws (the app encompasses all pets, but the ones with paws seem to go missing most often). pAwlert is also indicative of the alerts owners will receive when their pet is found, inducing a sentimental moment of ‘awww’ upon reunion. Hence, ‘pAwlert.’

The front end of the app uses Javascript for functionality and Handlebars.js to dynamically render content. The back end is built in Node.js using the Express.js framework and a MySQL database. To store our information, users contribute to an API we built when they input a lost pet, which dynamically renders onto the home page with an AJAX call and displays with Handlebars.js.
